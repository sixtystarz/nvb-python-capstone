# Project Title
The Student Study Tracker
## What it does
What this program does is allow for students to log their daily study sessions, track the cumaltive hours studying, and generate summary reports on the most studied subjects. What this achieves is giving students something tangible to their studying. Seeing what subjects they lack on and comparing it to how much time they spend studying on it can truly change how many students study
## How to run it
1. Open the notebook in Google Colab
2. Run all cells from top to bottom
3. Input the corresponding number of desired output.

## Skills demonstrated
1. Lists/Dictionaries
2. Functions
3. Loops
4. Strings


## Key finding (Option A) / What it produces (Options B, C, D)
Produces accurate study times hours and most studied subjects along with options to add more classes studied for.
